# 📚 Projetos

## 🎒 Mochila de Viagem

> Simulação de organização de uma mochila com limite de **23 kg**.

Funcionalidades principais:

* Adicionar itens e peso
* Verificar limite da mochila
* Exibir peso total e espaço restante
* Listar itens adicionados

---

## 🔢 Números Primos

> Programa que identifica números primos dentro de um intervalo informado pelo usuário.

Funcionalidades:

* Leitura de intervalo numérico
* Verificação de números primos
* Contagem de primos encontrados
* Exibição do menor e maior primo

---

## 📒 Agenda Telefônica

> Sistema simples de agenda utilizando **dicionário em Python**.

Funcionalidades:

* Cadastrar contato
* Buscar contato
* Excluir contato
* Listar contatos cadastrados

---

## 📚 Sistema de Biblioteca (POO)

> Sistema simples de biblioteca desenvolvido com **Programação Orientada a Objetos**.

Funcionalidades:

* Cadastro de livros
* Cadastro de usuários
* Empréstimo de livros
* Devolução de livros
* Listagem de livros disponíveis
